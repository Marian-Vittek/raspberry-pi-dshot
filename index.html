Raspberry Pi Dshot protocol implementation.

This is an implementation of DSHOT protocol for Raspberry Pi. It allows to control ESCs directly via the GPIO pins.


Download

You can download source code from github.


About

This implementation handles DSHOT 150 protocol well, other versions may cause problems. The software is running on standard Raspbian OS, no RT extension is required. However, it may help if you use some hacks reducing interrupts on a specific core.



The code is freely inspired by David Lawson's implementation. Compared to the original, this version differs in two points:
  • It uses "clock_gettime" for timing instead of an empty loop.
  • It allows multiple frames to be sent to multiple pins at once.
